# 🎓 JNTUA CEA Placement Analytics Dashboard

## 📌 Project Overview

This project is a Placement Analytics Dashboard developed for 
Jawaharlal Nehru Technological University Anantapur – College of Engineering, Anantapur (JNTUA CEA).

The dashboard provides year-wise, branch-wise, and company-wise placement insights using real institutional placement records.

Note:
- Branch and company data are based on actual placement records.
- Salary (LPA) values are sample/anonymized data used for visualization purposes only.
- 
## 🎯 Objectives

- Analyze placement trends across multiple academic years
- Compare branch-wise average salary packages
- Visualize company-wise student selection
- Provide an interactive and user-friendly analytics interface

## 🛠 Technologies Used

- Python
- Streamlit
- Pandas
- Plotly
- GitHub
- Streamlit Community Cloud

## 🚀 Key Features

- Dynamic year selection
- Branch-wise average package visualization
- Company-wise placement analysis
- Multi-year comparison dashboard
- Fully deployed cloud-based application
